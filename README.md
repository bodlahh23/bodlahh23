🔬 **Yeast Genome Analysis Pipeline**

- Simulated gene expression distributions

🧬 **DNA GC/AT Skew Analyzer (Python)**

Sliding-window genomic analysis pipeline computing:
- GC Content
- GC Skew: (G − C) / (G + C)
- AT Skew: (A − T) / (A + T)
Features:
- Modular analysis functions
- Divide-by-zero safe statistical calculations
- Matplotlib visualizations
- Extendable to FASTA-based genome-scale analysis

Concepts Demonstrated:
Nucleotide bias detection, strand asymmetry analysis, genomic signal visualization.

🧠 **Conway’s Game of Life Simulation**

Multi-generational cellular automaton built from scratch in R.
Features:

- State-transition modeling
- Grid-based simulation engine
- Dynamic visualization using ggplot2

Concepts Demonstrated:
Algorithmic modeling, simulation design, state transitions, computational systems thinking

🧬 **De Novo Genome Assembly Simulation**
